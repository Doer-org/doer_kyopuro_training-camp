## Python文法基礎講座
- 入力受け取り
入力受け取りでよく使う関数一覧
```
# 入力を受け取る関数
input
# 受け取った入力を空白区切りにする
split
# 文字列型を数字に変える
int 
# 複数の要素に同じ関数を使う
map

```
```
S = input()
```
```
S = int(input())
```
```
A, B = input().split()
```
```
A, B = map(int, input().split())
```
